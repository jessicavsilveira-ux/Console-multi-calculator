# Console Multi Calculator

A multi-function console calculator written in Portugol.

This program includes several utilities combined into a menu-based interface.

## Features

1. Basic calculator
2. Temperature converter
3. Data storage converter (bit, byte, KB, MB, GB, TB)
4. Acceleration converter
5. Time converter
6. Pregnancy calculator
7. Roman numeral converter

## Technologies

- Portugol
- Structured programming
- Menu-driven console interface

## Concepts Used

- Variables
- Conditional statements
- Loops
- Arrays
- Mathematical operations
- User input/output

## Example Menu
1 - Basic calculator
2 - Temperature converter
3 - Storage converter
4 - Acceleration converter
5 - Time converter
6 - Pregnancy calculator
7 - Roman numeral calculator
8 - Exit

## Purpose

This project was created to practice programming logic and algorithms using Portugol.

It demonstrates how to build a complex menu-driven system with multiple utilities in a single program.
